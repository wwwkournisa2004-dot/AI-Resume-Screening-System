# AI Resume Screening System using NLP

## Overview

This project is an AI-based Resume Screening System developed using Python, NLP, and Machine Learning. It compares candidate resumes with a job description and ranks candidates based on similarity scores.

## Features

- AI-based resume ranking
- Match percentage calculation
- Candidate recommendation
- Skill extraction
- Interactive job description input
- Resume search
- Bar chart visualization
- Pie chart visualization

## Technologies Used

- Python
- Google Colab
- Pandas
- Scikit-learn
- Matplotlib
- NLTK
- TF-IDF Vectorizer
- Cosine Similarity

## Project Workflow

1. Load resume dataset
2. Enter job description
3. Convert text using TF-IDF
4. Calculate cosine similarity
5. Rank resumes
6. Display best candidate
7. Generate charts
8. AI hiring recommendation
